<h1 align="center"> 期末專案說明 </h1>

建置一套服務系統 ( 題目自訂，須有明確應用主題，如 橘子商場、寵物美容等)，須有下列功能： 

1. 會員註冊( 寄信開啟帳號 ) 、有登入登出。 

2. 需有上傳資料及圖片功能 (例如電子商務網站的商品上架 ) 

3. 需有管理服務內容的功能 (例如電子商務網站的商品管理 ) 

4. 需有顧客介面 (購物車、訂貨(預約)、等功能。請妥善設計顧客瀏覽商品或服務的頁面，提供顧客流暢的消費介面。 

5. 需有訂單及會員管理介面及功能 (查看訂單、查詢訂單、訂單統計(含統計圖))。這個功能是給店家管理員使用的。 

請根據你的服務內容設計界面，愈接近真實愈好。服務可以是銷售物品，也可以是提供服務如寵物美容預約、美甲預約等) 

原則上以全端網站作為開發平台(即全部以 PHP 完成 )。完成後若有改寫成前後端分離 或 Laravel 架構，另外加分！ 

  

### 評分標準 

1. 系統流暢及完整性 demo 

2. 系統構成的架構說明 (是否有架構圖並能說明 ) 

3. 程式碼理解考核 ( 依照程式執行邏輯考核同學的程式碼理解程度 ) 

4. 應用創意性 ( 5% )
