# 個人網站期末專案架構

這個資料夾是依照 PDF 作業要求建立的網站骨架。

## 資料夾

- `index.html`: 網站首頁，包含全部主要區塊。
- `css/styles.css`: 手寫 RWD 樣式，已包含 `@media`。
- `js/main.js`: 手機導覽與目前區塊高亮效果。
- `images/`: 本人照片、專案截圖、活動照片放這裡。

## 要替換的內容

1. 將資料夾名稱 `學號` 改成你的正式學號。
2. 將 `index.html` 裡的 `姓名 / 學號` 改成你的資料。
3. 把 `images/profile-placeholder.svg` 換成本人照片。
4. 把專案作品、文章、獲獎與活動照片改成你的真實內容。
5. 部署到 GitHub Pages 後，把頁面中的 GitHub Pages 網址改成正式網址。
6. 錄製 3 分鐘內說明影片，填入老師提供的表單。

## 作業提醒

- 上傳檔案名稱需為 `學號.zip`。
- 網站目錄需為 `學號`。
- 圖片子目錄需為 `images`。
- 檔案、個人網頁網址、說明影片網址三者都要提供。
